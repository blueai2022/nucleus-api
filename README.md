# nucleus-api

## Objectives:

Coding agent has come a long way using MCP to retrieve context live for coding purposes. However, accurate context building and requirements definitions are still largely manual processes. The project aims to use Agentic AI Workflow to automate the processes of such tasks with a "Coding Agent as Backend" approach. 

It is intended to offer development tools to SWEs more from a workflow point of view: with Coding Agent performing a series of automated tasks to produce context artifacts and validate them so that they can readily used and trusted.

## Current State:

Creating an Agentic AI Workflow to have some basic skills.
